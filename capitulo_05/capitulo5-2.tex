\section{Configuración de VPN}

La especificación para PPTP fue publicada por el RFC 2637\cite{rfc:2637}, aunque no ha sido ratificada como estándar por el IETF.\\

Point-To-Point Tunneling Protocol (\Gls{pptp}) permite el intercambio seguro de datos de un cliente a un servidor formando una Red Privada Virtual (VPN, por el anglicismo Virtual Private Network), basado en una red de trabajo vía \Gls{tcp/ip}. El punto fuerte del PPTP es su habilidad para proveer en la demanda, multi-protocolo soporte existiendo una infraestructura de área de trabajo, como INTERNET. Esta habilidad permitirá a una compañía usar Internet para establecer una red privada virtual (VPN) sin el gasto de una línea alquilada.

Ahora agregamos nuestros Clientes VPN, Para eso nos vamos a la pestaña Secrets, y agregamos los usuarios que sean necesarios.\\

\textbf{Name:} Nombre de Usuario VPN\\
\textbf{Password:} Contraseña del Usuario VPN\\
\textbf{Service:} Tipo de Protocolo que usara el cliente para su coneccion con el servidor.\\
\textbf{Profile:} Perfil de encryptacion \\
\textbf{Local Address:} Dirección IP del equipo anfitrión, del router local (La IP de puerta de enlace de nuestra LAN anfitrión.)\\
\textbf{Remote address:} Direccion IP con la cual el equipo cliente se identificara en nuestra re LAN.\\
